# CyberToolkit

A modular cybersecurity and network analysis toolkit written in Python.

## Features

### Recon
- IP Information
- DNS Lookup
- Whois Lookup

### Web Analysis
- SSL Analyzer
- HTTP Header Analyzer
- Technology Detector

### Network
- Port Scanner

### Crypto
- Hash Generator
- Hash Identifier
- JWT Decoder

### Utilities
- Base64 Encoder/Decoder
- URL Parser
- HTML Report Generator
- CSV Exporter
- Statistics Dashboard
- Settings Manager

## Installation

```bash
git clone https://github.com/yourusername/cybertoolkit.git

cd cybertoolkit

pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Project Structure
